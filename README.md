
# installation
## clone the repository
 ```
 git clone https://github.com/kayancms/maritime.git
 ```

## install  requirements 
```
composer install
npm install 
```

## database migration and seeding
```
php artisan migrate --seed
```


## running
```
php artisan serv 
npm run dev
```
